═══════════════════════════════════════════════════════════════
⚠️  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Build fails?
  1. Check .netrc exists in project root
  2. Verify gradle.properties has MAPBOX_DOWNLOADS_TOKEN
  3. Run: eas build --clear-cache

Map doesn't show?
  1. Check internet connection
  2. Verify Mapbox token at https://account.mapbox.com/
  3. Test on real device (won't work in Expo Go)

Location not working?
  1. Grant location permissions in settings
  2. Enable GPS on device
  3. Test outdoors for better signal
