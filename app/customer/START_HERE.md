## 🚀 Build for Play Store (5 Steps)

### Step 1: Install EAS CLI
```bash
npm install -g eas-cli
```

### Step 2: Login
```bash
eas login
```
*Create account at https://expo.dev if you don't have one*

### Step 3: Go to Project
```bash
cd /home/sakthi-selvan/jk_taxi/app/customer
```

### Step 4: Install Dependencies
```bash
npm install
```

### Step 5: Build!
```bash
eas build --platform android --profile production
```

**That's it!** Your AAB will be ready in 15-20 minutes.
